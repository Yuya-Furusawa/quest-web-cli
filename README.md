# quest-web-cli

Quest App の Web フロントエンド、React + TypeScript + Vite で書く

# 開発環境の構築
## `.env`ファイルの作成
ルートディレクトリに `.env` を作成し、別でもらった内容を追加する

## nodenvのインストール
https://zenn.dev/donchan922/articles/b08a66cf3cbbc5

## nodeのインストール
```
nodenv install 18.3.0
node rehash
```

## nodeのバージョン指定
```
nodenv local 18.3.0
```

## パッケージインストール
```
npm install
```

## run
```
npm run dev
```
http://localhost:5173/
